# vue-storybook-sample

## 前提

- `@vue/cli@3.4.1`
- `@vue/cli-service@3.4.1`

## セットアップ手順

```bash
yarn
```

### 普通に起動する場合
```bash
yarn serve
```

### storybookを起動する場合
```bash
yarn storybook:serve
```
